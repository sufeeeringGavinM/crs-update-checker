# crs-update-checker
Checks UPD CRS Regular Classes regularly for updates. Saves as html.

# Installation (Python)

1. Install requirements.txt
2. Download the .py.

# Installation (Normal Human)
1. Download the .exe

# Usage
Wherever the .py or the .exe file is placed, a folder will be created and that's where all the .csv and .html files will be made. Just open the .py/.exe and it'll run a scan of the class schedules regularly, comparing the current and previous scan. Whenever some change happens, you can see it in LatestChange.html
